# PrismUserControlTest

## Description

Sample application for generating collection of custom UI elements with dynamic property in Prism WPF.

## Requirement

- .NET Framework 4.7.1
- Prism.Wpf 7.1.0.431
- Prism.Unity 7.1.0.431

## Usage

Build this with Visual Studio 2019 and run.
